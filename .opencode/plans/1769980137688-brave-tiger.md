# Landing Page Plan - PlatoSano (CENAN Meal Planner)

## Overview
Build a landing page for **PlatoSano** - a meal planning app that helps Peruvian families plan weekly meals based on nutritious, low-cost regional recipes from Peru's Instituto Nacional de Salud (CENAN).

## Brand
- **Name:** PlatoSano
- **Tagline:** "Comidas saludables para tu familia"

## Target Audience
- Peruvian families, primarily homemakers (amas de casa)
- Budget-conscious households
- People who want to eat healthier without drastic changes

## Design System (Based on Fitia Reference)

### Colors
| Variable | Hex | Usage |
|----------|-----|-------|
| `--background` | `#FDF8F3` | Warm cream page background |
| `--background-white` | `#FFFFFF` | Cards, clean sections |
| `--foreground` | `#1A1A1A` | Primary text |
| `--primary` | `#E5A93D` | Golden yellow CTAs |
| `--primary-hover` | `#D4922E` | Button hover state |
| `--text-secondary` | `#6B7280` | Descriptions |
| `--card-yellow` | `#FEF3C7` | Feature cards |
| `--card-green` | `#D1FAE5` | Feature cards |
| `--card-purple` | `#E9D5FF` | Feature cards |
| `--card-blue` | `#DBEAFE` | Feature cards |

### Typography
- Headlines: Geist Sans Bold, 36-48px
- Body: Geist Sans Regular, 16-18px
- All content in Spanish

### Components
- Buttons: `rounded-full`, golden yellow primary
- Cards: `rounded-2xl`, no borders, pastel backgrounds
- Emojis as decorative icons
